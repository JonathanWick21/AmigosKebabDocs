# Reusable Card

Esta es la documentación **narrativa** del componente de detalles. 
Se utiliza para estandarizar cómo mostramos los productos de **Amigos Kebab**.

### ¿Cuándo usarlo?
Cuando necesites mostrar un producto en el catálogo principal o en la sección de destacados.

## Propiedades
Usa los controles en las historias individuales para probar diferentes estados (precio, textos largos, etc.)

Ver las historias en la pestaña "Canvas" para interactuar con el componente.
